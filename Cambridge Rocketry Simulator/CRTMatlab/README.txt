The Cambridge Rocketry Toolbox is a FREE rocket flight simulation toolbox for Octave.

To use the toolbox you will need Octave 3.0 or greater

To install octave on your machine:

Windows users - Download from http://octave.sourceforge.net/

Linux users - From repositories, or from source at http://www.gnu.org/software/octave/. In addition you will need to install the forge packages from http://octave.sourceforge.net/

To use the toolbox either add this folder to your Octave path or make it the 'active' folder. For instructions on how to use the toolbox please see the files CRT_InstructionManual.pdf and CRT_Tutorials.pdf in this folder.
